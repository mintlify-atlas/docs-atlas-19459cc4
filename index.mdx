Launchpad is an open source suite of tools that help people and teams work together on software projects. It powers [launchpad.net](https://launchpad.net), Canonical's hosted collaboration platform used by thousands of Ubuntu and open source projects worldwide.

## What you can do with Launchpad

<CardGroup cols={3}>
  <Card title="Bug Tracking" icon="bug" href="/features/bugs">
    Track bugs, link to upstream trackers (GitHub, Bugzilla, GitLab), and manage bug lifecycle across projects and distributions.
  </Card>
  <Card title="Code Hosting" icon="code-branch" href="/features/code-hosting">
    Host Git repositories, manage branches, and run code reviews through merge proposals with inline diffs and comments.
  </Card>
  <Card title="Package Publishing" icon="box" href="/features/packages">
    Publish Ubuntu packages via Personal Package Archives (PPAs) and manage distribution uploads and binary builds.
  </Card>
  <Card title="Translations" icon="language" href="/features/translations">
    Crowdsource software translations using the Rosetta platform, with support for PO files and translation imports/exports.
  </Card>
  <Card title="Blueprints" icon="map" href="/features/blueprints">
    Plan features and specifications with blueprint tracking, priority assignments, and milestone targeting.
  </Card>
  <Card title="Build Services" icon="hammer" href="/features/builds">
    Build Snap packages, Charm operators, OCI images, and Debian packages across multiple architectures.
  </Card>
</CardGroup>

## Get started

<Steps>
  <Step title="Create your Launchpad account">
    Visit [launchpad.net](https://launchpad.net) and sign in with your Ubuntu One account. If you don't have one, you can register for free.
  </Step>
  <Step title="Create or join a project">
    Register a new project or search for an existing one. Projects are the top-level container for your bugs, code, translations, and blueprints.
  </Step>
  <Step title="Set up your first repository">
    Import or create a Git repository for your project. Launchpad supports both Git and the legacy Bazaar VCS.
  </Step>
  <Step title="File a bug or start translating">
    Use the bug tracker to report issues, or dive into Rosetta to start contributing translations. The REST API lets you automate both.
  </Step>
</Steps>
